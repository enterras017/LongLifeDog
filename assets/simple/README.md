# ながいぬスネークゲーム用画像ファイル

## 必要な画像ファイル

以下の画像ファイルを `assets/simple/` ディレクトリに配置してください：

### 既存ファイル（使用中）
- `longdog_head.png` - 通常の顔
- `longdog_body.png` - 胴体
- `longdog_tail.png` - 尻尾

### 追加が必要なファイル
- `longdog_head_smile.png` - 笑顔の顔
- `longdog_head_sad.png` - 悲しい顔
- `food.png` - ごはん（骨やお肉の画像）
- `bg_grass.png` - 背景（草原）

## 画像の仕様
- サイズ: 30x30px 程度
- 形式: PNG（透明背景推奨）
- 色調: 明るくてかわいい色合い

## 現在の代替表示
- 笑顔・悲しい顔: 通常の顔画像を使用
- ごはん: 🍖 絵文字で表示
- 背景: 緑色の単色背景

画像ファイルを追加後、SnakeGame.tsx の該当部分を更新してください。
