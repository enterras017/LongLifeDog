# リリースノート v1.0.0

**リリース日**: 2025年11月9日  
**バージョン**: 1.0.0 (Build 1)  
**プラットフォーム**: iOS 13.0+

---

## 🎉 初回リリース - MVP完成！

「ながいぬのいる生活」の最初のバージョンをリリースします！

ご飯をあげると体がニョキニョキ伸びる不思議な生き物「ながいぬ」を育てる癒し系育成ゲームです。

---

## ✨ 新機能

### 🐕 メイン画面
- **ご飯をあげる機能**: ながいぬにご飯をあげて体を伸ばそう
- **なでなで機能**: ながいぬの頭をなでなでして癒されよう
- **表情変化**: 通常・笑顔・悲しい顔の3パターン
- **効果音**: なでなですると「わん！」と鳴きます
- **データ自動保存**: ながいぬの成長記録を自動保存

### 🎮 ご飯ランナー（ミニゲーム）
- **スワイプ操作**: 画面をスワイプしてながいぬを操作
- **スピードアップシステム**: ご飯を集めるとスピードが上がる
- **ハイスコア記録**: 最高記録を自動保存
- **効果音＋バイブレーション**: ご飯を取得時とゲームオーバー時

### 🎓 チュートリアル
- **初回起動時**: メイン画面の使い方を説明
- **初回プレイ時**: ご飯ランナーの遊び方を説明
- **いつでも再表示**: ？ボタンでいつでもチュートリアルを確認可能

### ⚙️ 設定画面
- **効果音ON/OFF**: 音声のON/OFF切り替え
- **バイブレーションON/OFF**: 振動のON/OFF切り替え
- **チュートリアルリセット**: 初回チュートリアルを再表示
- **アプリ情報**: バージョン情報の確認

### 💾 データ永続化
- **自動保存**: すべてのデータを自動的に保存
- **保存内容**:
  - ながいぬの体の長さ
  - ご飯をあげた回数
  - なでなでした回数
  - ご飯ランナーのハイスコア
  - 設定情報

## 🎯 インストール方法

### App Store（準備中）
1. App Storeで「ながいぬのいる生活」を検索
2. 「入手」ボタンをタップ
3. インストール完了後、アプリを開く

### 開発版（開発者向け）
```bash
git clone https://github.com/enterras017/LongLifeDog.git
cd LongDogLifeNew
npm install
cd ios && pod install && cd ..
npx react-native run-ios
```
